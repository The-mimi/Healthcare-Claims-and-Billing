# Healthcare Claims and Billing Insights Using SQL
**Objective:**
<p>Evaluate the operational efficiency and financial performance of healthcare claims to identify patterns in processing times, denial rates, and cost discrepancies, guiding strategies for process improvement and cost reduction.</p>

**Key Insights**
- Average Processing Time: 23 days
- Main Delay: “Under Review” stage (+2.4 days above average)
- Approval Rate: 55.3% for in-network vs. 9.1% for out-of-network
- Top Denial Reason: “Not medically necessary” (23%)
- Peak Denial Month: December (12.5%)
- Average Patient Responsibility: $179
- Provider Revenue Loss: $113,313 due to underpayments
  
**Dataset Overview:**
The healthcare claims dataset contains:
- ~4,000+ claims with dates, claim status (paid, denied, pending), and network status(in-network or out-of-network). 
- Operational Date: Denial reasons and priority flags for processing urgency.
- Financial Data: Billed amounts charged by providers, allowed amounts approved by payers, amounts paid by insurer, and member responsibilities(co-pays, deductibles).

**Tools Used**: SQL (BigQuery), Power BI, Excel
**<p>Focus Areas**: Processing time efficiency, Financial accuracy, Denial management, and Network impact.</p>

**Deliverables**
- SQL Analysis
- Full Report: data prep, correlation modeling, and visual outputs.
- Interactive Dashboard: Performance and Financial insights.​
